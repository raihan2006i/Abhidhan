Anubad Abhidhan is a E2B (English to Bangla) and B2E (Bangla to English) dictionary. The main feature of this dictionary is – you can search English or Bangla word in any word documents, pdf  documents , website contents by selecting the word and pressing Ctrl+C. Result will be shown on a Balloon Pop Up. It will be a great application for you if you use it! This application is free
